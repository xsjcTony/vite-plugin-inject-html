## Supported Options

- `favicon` - favicon url
- `title` - title
- `metas` - `<meta>` tags
- `links` - `<link>` tags
- `externalStyleSheets` - external style sheet urls
- `scripts` - `<script>` tags
- `noscripts` - `<noscript>` tags
- `otherTags` - other tags or custom `injectTo` preference
